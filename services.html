React JS

React.js is a powerful JavaScript library for building dynamic and interactive user interfaces, allowing developers to create fast, scalable, and maintainable web applications.

Node JS

Node.js is a JavaScript runtime built on Chrome's V8 engine that enables server-side scripting, allowing developers to build scalable, fast, and efficient network applications.

Express JS

Express.js is a minimal and flexible Node.js web application framework that provides robust features for building APIs and web applications, simplifying routing, middleware, and server-side logic.

My SQL

MySQL is an open-source relational database management system (RDBMS) that uses structured query language (SQL) to manage and manipulate data, providing a reliable and efficient platform for storing and retrieving data in web applications.

HTML 5

HTML5 is the latest version of the HyperText Markup Language, designed to structure and present content on the web, offering new features like multimedia support, semantic elements, and improved performance for modern web applications.

CSS 3

CSS3 is the latest version of Cascading Style Sheets, providing advanced styling capabilities for web pages, including animations, transitions, flexible layouts, and responsive design to create visually appealing and adaptive user interfaces.
